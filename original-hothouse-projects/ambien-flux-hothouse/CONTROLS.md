# Ambien Flux - Quick Controls Reference

## Global Controls (Always Active)
- **K1**: Master Level (0-200%)
- **K2**: Dry/Wet Mix (0-100%)

## Toggle 3 UP - Normal Mode
- **K3**: Feedback (0-100%)
- **K4**: Slice Count (1-16)
- **K5**: Slice Length (100-500ms)
- **K6**: Stutter (0-100%)

## Toggle 3 DOWN - Lo-Fi Mode
- **K3**: Wobble (tape flutter)
- **K4**: Dust (vinyl crackle)
- **K5**: Bit Crush (sample rate reduction)

## Toggle 1 - Playback Modes
- **UP**: Forward/Forward
- **MIDDLE**: Reverse/Reverse
- **DOWN**: Forward/Random

## Footswitches
- **FS1 TAP**: Bypass
- **FS1 HOLD** (2s): Bootloader
- **FS2 TAP**: Freeze

## LEDs
- **LED1**: Effect active
- **LED2**: Freeze active
